# 🏡 AirTours Frontend – Airbnb Clone UI

AirTours Frontend is a modern Airbnb-inspired UI built with React/Next.js. It focuses on clean design, responsiveness, and smooth user experience.

## 🚀 Features
- Property listing UI
- Search & filter interface
- Booking date picker UI
- Wishlist/Favorites UI
- Fully responsive design
- Modern UI with Tailwind CSS

## 🛠️ Tech Stack
- React.js / Next.js
- Tailwind CSS
- TypeScript (optional)

## 📂 Folder Structure
```
src/
├── components/
├── pages/
├── assets/
└── styles/
```

## ⚙️ Setup
```
git clone https://github.com/your-username/airtours-frontend.git
cd airtours-frontend
npm install
npm run dev
```

## 🌐 Local Server
http://localhost:3000
